2. Register the Debezium PostgreSQL connector. Port-forward to the Connect
   REST API and POST the connector config:

   kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "pg2mongo-cdc.fullname" . }}-connect 8083:8083 &

   curl -X POST -H 'Content-Type: application/json' \
     -d @connectors/debezium-postgres.json \
     http://localhost:8083/connectors

   curl -fsS http://localhost:8083/connectors?expand=status \
     | jq '.[] | {name: .status.name, state: .status.connector.state, tasks: [.status.tasks[].state]}'

3. Smoke test by inserting a row in Postgres and confirming it appears in
   MongoDB within 5 seconds. See docs/deployment.md "Smoke test".

4. Wire up the Prometheus alerts from observability/prometheus/alerts.yml.
   Each alert maps to a section in docs/runbook.md.

5. Verify the SLOs (docs/slo.md) on the Grafana dashboard
   "Migration Overview".
